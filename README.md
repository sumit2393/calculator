# String Calculator (TDD with Flutter & Provider)

## Getting Started

A simple String Calculator implemented using Test-Driven Development (TDD).
The project demonstrates clean separation of business logic and UI using Flutter
and Provider for state management.

## Problem Statement
The String Calculator accepts a string of numbers and returns their sum.
It supports multiple delimiters, newline-separated values, and validates
negative numbers by throwing descriptive exceptions.

## Features
- Add numbers from a string input
- Supports unlimited numbers
- Supports newline (`\n`) as a delimiter
- Supports custom delimiters (e.g. `//;\n1;2`)
- Throws exception for negative numbers
- Displays all negative numbers in error message
- Clean UI built with Flutter
- State management using Provider
- Business logic fully unit-tested (TDD)

## Tech-Stack
- Flutter
- Dart
- Provider (State Management)
- flutter_test (Unit & Widget Testing)
